# plane_in_medical

#### 介绍
医飞速达是一个基于Django框架的医疗无人机配送系统。项目结合了强化学习算法和Web开发技术，实现了智能路径规划和多医院协同库存管理。系统包含用户认证、商品浏览、购物车、订单管理、在线支付等完整电商功能，并通过强化学习优化配送路径，提高配送效率。前端采用响应式设计，后端使用MySQL数据库，支持多语言切换和用户个性化设置。
